GRAVITY - GitHub Project
Stephen Hicks

A simple orbital simulator. The goal is to dock your vessel (green) with the target (red). Controls are as follows:

(Shift+)W(Small) Thrust Forward
(Shift+)S(Small) Thrust Backward
(Shift+)A(Small) Turn Left
(Shift+)D(Small) Turn Right
(Shift+)Q(Small) Thrust Left
(Shift+)E(Small) Thrust Right

(Shift+)0Random Ship Position
(Shift+)9Random Target Position
(Shift = more circular)
?Display Help
XToggle Stats Visibility

[/]Speed Up/Down Clock
=Reset Speed
SpacePause

ZToggle Planet/Ship View
LToggle Line Visibility
+/-Zoom In/Out (Ship View only)
=Reset Zoom
